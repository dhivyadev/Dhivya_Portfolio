Figma
Cloud-based design and prototyping tool used by designers and teams to create user interfaces, web designs, mobile app designs, and other digital designs collaboratively.
Canva
Web-based graphic design and visual content creation tool that offers a user-friendly interface and a wide range of templates for creating various types of designs, including social media graphics, presentations, posters, flyers, and more.
Bootstrap5
Open-source Front-end framework developed by Twitter that provides a collection of pre-designed and reusable HTML, CSS, and JavaScript components includes a wide range of UI elements.
(i.e.,such as buttons, forms, navigation bars, modals, and more, all styled and optimized for consistent and attractive design.)
Visual Studio Code
Free and open-source Code editor developed by Microsoft. It is highly popular among developers for its versatility and extensive extension ecosystem.
Pycharm Community Edition
Free and open-source Integrated Development Environment (IDE) specifically tailored for Python programming. Developed by JetBrains, offers a range of features to assist Python developers in writing, debugging, and testing their code efficiently.
Reactjs
Open-source JavaScript library for Building User Interfaces (UIs) and web applications. Developed and maintained by Facebook, React focuses on creating highly efficient and interactive UI components. It employs a component-based architecture, where developers can build UI elements
Nextjs
Open-source React framework for building Web applications. It is designed to simplify and streamline the development of server-rendered React applications, providing features and tools to help developers build fast, scalable, and SEO-friendly websites and web applications.
Nodejs
Open-source, Server-side JavaScript runtime environment that allows developers to execute JavaScript code outside of a web browser. It is built on the V8 JavaScript engine and is designed for building scalable and high-performance network applications.
NOSQL WorkBench
NoSQL Workbench is a comprehensive software tool Developed by Amazon Web Services (AWS) designed to facilitate the design, development, and querying of NoSQL databases.
MySQL
Open-source Relational Database Management System (RDBMS) that is widely used for storing and managing structured data. Developed by Oracle Corporation,compatible with multiple operating systems and programming languages.
MongoDB
MongoDB is a popular, open-source NoSQL Database Management System that is designed to handle unstructured or semi-structured data, stores data in a flexible, document-oriented format using BSON (Binary JSON).
Amazon Web Services (AWS)
Comprehensive and widely used Cloud computing platform and Infrastructure service offered by Amazon.com AWS provides a vast array of cloud-based services, including computing power, storage, databases, machine learning, analytics, content delivery, Internet of Things (IoT), security, and more.
Google Services
Cloud Computing platform and suite of Cloud-based services offered by Google. It provides a wide range of Infrastructure, Data storage, Machine learning, Analytics, and Application Development tools that enable businesses and developers to build, deploy, and manage applications and services in the cloud.
Azure
Cloud computing platform and Infrastructure service offered by Microsoft. It provides a comprehensive suite of cloud-based services, including computing, storage, databases, networking, analytics, machine learning, and more, to help organizations build, deploy, and manage applications and services in the cloud.
Global Information Tracker (GIT)
Open-source distributed Version Control System that allows developers to track changes in their codebase, collaborate on software projects, and manage different versions of their code efficiently. Developed by Linus Torvalds in 2005, Git has become an essential tool in the software development industry.
Docker
Open-source platform that simplifies the process of Developing, Deploying, and Managing applications within containers. Containers are lightweight, portable, and isolated environments that package an application and its dependencies.
